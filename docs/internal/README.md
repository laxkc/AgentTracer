## Platform Capabilities

AgentTracer provides three integrated capabilities:

1. **Execution Observability** - Agent runs, steps, latency, failures
2. **Decision & Quality Observability** - Decision points, quality signals
3. **Behavioral Drift Detection** - Baselines, statistical comparison, alerts

All capabilities are production-ready (January 2026).
